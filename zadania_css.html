Назад
При стилизации с помощью CSS можно редактировать как каждый отдельный класс или элемент, задавая ему имена с помощью class="" или id="", так и элементы целиком, такие как <body>, <main>, <input> и так далее
Именование с помощью class нужно если к нескольким элементам необходимо применить одинаковую стилизацию
Пример: 
                .description {
                    style
                }
Именование с помощью id нужно если необходимо применить стилизацию только к одному конкретному элементу
Пример: 
                #picture {
                    style
                }
При изменении вида элемента, стилизация применяется во всем элементам этого типа и вложенным в них
Пример: 
                body {
                    style
                }

Пункт 3-ий
Для группы элементов .top, .bot, .footer-bottom зададим атрибуты display: flex (Элемент генерирует структурный блок, который создает адаптивный контейнер для дочерних элементов.) и flex-flow: column (построение элементов колонной) nowrap (запрет на смещение элементов)
Для группы элементов .top, .footer-bottom зададим атрибуты align-items: center (смещение элементов по центру); text-align: center (смещение текста по центру)
Для элемета .top задать параметры background-image: url("путь к картинке фона"); background-repeat: no-repeat (отмна повтора картинки); background-size: cover (масштабирование изображения по заданным размерам);
Для элемента footer задайте атрибуты font-size (размер шрифта) равным 5mm
Для элемента #picTPK задайте атрибуты padding(отступы снаружи): 5mm(сверху) 0(снизу) 10mm(справа) 0(слева);
Для элемента #TPKdesc задайте атрибут font-family(шрифт): 'Source Sans Pro', sans-serif;;
