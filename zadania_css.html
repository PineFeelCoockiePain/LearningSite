Задание №2

При стилизации с помощью CSS можно редактировать как каждый отдельный класс или элемент, задавая ему имена с помощью class="" или id="", так и элементы целиком, такие как <body>, <main>, <input> и так далее
Именование с помощью class нужно если к нескольким элементам необходимо применить одинаковую стилизацию
Пример: 
                .description {
                    style
                }
Именование с помощью id нужно если необходимо применить стилизацию только к одному конкретному элементу
Пример: 
                #picture {
                    style
                }
При изменении вида элемента, стилизация применяется во всем элементам этого типа и вложенным в них
Пример: 
                body {
                    style
                }
В html файл вашего сертификата импортируйте css файл в нижнюю часть <head> 
                <link rel="stylesheet" href="sertificate.css">
Осталось только его отредактировать
Импортируйте шрифты с помощью данной функции 
                @import url('https://fonts.googleapis.com/css2?family=Raleway:wght@200&family=Roboto+Condensed:wght@700&family=Source+Sans+Pro:wght@300;400&display=swap');
Зададим ширину и высоту страницы (элемент body) под размеры листа A4: ширина 210mm, высота 297mm
Также зададим градиентный фон: вместо белого зададим линейный градиент от низа к верху, функцией: 
                linear-gradient(to top, #9696fb, #f8f9fb, #f8f9fb)
Для элементов .top, .bot, .footer-bottom зададим значение display: flex (Элемент генерирует структурный блок, который создает адаптивный контейнер для дочерних элементов.) и flex-flow: column (построение элементов колонной) nowrap (запрет на смещение элементов)
Для элементов .top, .footer-bottom зададим значения align-items: center (смещение элементов по центру); text-align: center (смещение текста по центру)
Для элемета .top задать параметры background-image: url("путь к картинке фона"); background-repeat: no-repeat (отмна повтора картинки); background-size: cover (масштабирование изображения по заданным размерам);
