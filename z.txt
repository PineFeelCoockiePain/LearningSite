Задание 2: стилизация CSS
При стилизации с помощью CSS можно редактировать как каждый отдельный класс или элемент, задавая ему имена с помощью class="" или id="",
так и элементы целиком, такие как <body>, <main>, <input> и так далее
Именование с помощью class нужно если к нескольким элементам необходимо применить одинаковую стилизацию
Пример:
.description {
style
}
Именование с помощью id нужно если необходимо применить стилизацию только к одному конкретному элементу
Пример:
#picture {
style
}
При изменении вида элемента, стилизация применяется во всем элементам этого типа и вложенным в них
Пример:
body {
style
}
Если вы хотите использовать шрифты с google fonts, нужно импортировать его в начале файла
С помощью функции @import url('ссылка')
Пример:
@import url('https://fonts.googleapis.com/css2?family=Montserrat:wght@200&family=Open+Sans+Condensed:wght@300&family=Roboto+Condensed:wght@700&display=swap');
