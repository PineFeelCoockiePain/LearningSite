Задание 2: стилизация CSS
При стилизации с помощью CSS можно редактировать как каждый отдельный класс или элемент, задавая ему имена с помощью class="" или id="",
так и элементы целиком, такие как <body>, <main>, <input> и так далее
Именование с помощью class нужно если к нескольким элементам необходимо применить одинаковую стилизацию
Пример:
.description {
style
}
Именование с помощью id нужно если необходимо применить стилизацию только к одному конкретному элементу
Пример:
#picture {
style
}
При изменении вида элемента, стилизация применяется во всем элементам этого типа и вложенным в них
Пример:
body {
style
}
В html файл вашего сертификата импортируйте css файл в нижнюю часть <head>
<link rel="stylesheet" href="sertificate.css">
Осталось только его отредактировать
Импортируйте шрифты с помощью данной функции [потом изменю шрифты, но путь пока будет так]

Поместите его в самый верх файла
Так как поля и отступы элемента не являются обязательными, по умолчанию их значение равно нулю. Тем не менее, некоторые
браузеры добавляют этим свойствам положительные значения по умолчанию на основе своих таблиц стилей.
Очистить стили браузеров для всех элементов можно при помощи универсального селектора:
* { <!--Поместить под импортом шрифтов-->
    margin: 0;
    padding: 0;
}

Для элементов .top, .bot, .footer-bottom зададим значение display: flex (Элемент генерирует структурный блок, который создает адаптивный контейнер для дочерних элементов.) и flex-flow: column (построение элементов колонной) nowrap (запрет на смещение элементов)
Для элементов .top, .footer-bottom зададим значения align-items: center (смещение элементов по центру); text-align: center (смещение текста по центру)
